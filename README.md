# Teach Smart 
- "Empowering Educators, Enriching Minds."

- Small Ai based application that will help teachers to revise their topic on last minute.



## Technologies Used :

- Github, Nextjs, Tailwind, shadcn-ui, vercel,open-ai,text-davinci
